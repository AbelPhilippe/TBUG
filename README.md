## Clone repository

``` bash
git clone https://github.com/AbelPhilippe/TBUG
```
```bash
cd TBUG
```

## Use VENV

```bash
python3 -m venv venv
```
```bash
source venv/bin/activate
```

## Install requirements.txt

```bash
pip install -r requirements.txt
```

## Execute TBUG (UBUNTU / DEBIAN / KALI / WSL)

```bash
python3 tbug.py -u https://example.com
```
